Import  Hex Text command

 

This command reads a simple text file containing hex digits (such as one created with the Export Hex Text command) and converts it to binary.  The data is inserted at the current cursor location.  If the window is in overtype mode then you are prompted to enter insert mode.  The command is unavailable if the file is read-only.  The standard file open dialog is used to allow you to select a file to insert. The inserted bytes are selected so you can easily see which bytes were added.

 

A simple hex file provides little error checking, except that non-hex alphabetic characters generate an error.  Other non-hex characters are ignored.  A single hex digit not followed by another hex digit generates one byte; otherwise pairs of hex digits are used to generate one data byte.  For example, the text "1  2  3" would generate three byte values 01, 02, 03; whereas "123" would generate just two 12, 03.
