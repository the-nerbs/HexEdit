Save command (File menu)

 

Use this command to save the active file using its current name and directory. When you attempt to save a newly created file that has not yet been saved, HexEdit displays the Save As dialog box so you can name the file. If you want to change the name and/or directory of an existing file, before you save it, choose the Save As command.

 

HexEdit allows you to create backup copies of files. Every time the file is saved the previous version of the file, if any, is saved with the same name except that the file extension is changed to “.BAK”. (If the file being saved has an extension of “.BAK” then the previous version is saved with an extension of “.BAC”.) The second most recent version, if any, is lost. Whether or not backup files are created is determined by setting options in the System Options page.

 

The save operation may take a long time due to the following reasons:

  • you have inserted or deleted bytes near the start of a large file - HexEdit must move a large part of the file internally

  • you have chosen to create a backup of a large file - HexEdit must make a copy of the file's data

  • you have performed operations on very large selections - HexEdit may have created large temporary files that need to be copied into the saved file
