Open Special command (File menu)

 

This command opens the Open Special dialog which allows you to select a disk device to be opened.

 

You can open any raw disk device including optical media, floppy disks, flash disks, and disks/partitions formatted with any filesystem or not formatted at all.  You can also open any volumes that use a filesystem that is supported by the operating system - each volume is usually associated with a drive letter unless it has not been mounted.

 

Disk are edited almost identically to normal files, with a few small exceptions.  First wherever you would normally see a filename a name that describes the device is shown, and where you would see the location (ie folder or directory) you will see the raw device name.  For example, see the Recently Opened Files Dialog.

 

Also in the File Properties Page of the Properties dialog you will see other relevant information rather than file type information (last modified time etc).  Note that in the Summary Page you can enter comments etc as normal.

 

You can not change the length of a device when editing so Insert Mode is disabled.
