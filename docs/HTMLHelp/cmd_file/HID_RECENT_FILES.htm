All Recent Files (File menu)

 

HexEdit keeps track of all the files that you have opened. You can quickly open the last one to 16 files that you have opened directly from the File menu (see Recently Opened Files).

 

This command is more powerful as it allows you to see a list of all files that you have opened in the Recently Opened Files Dialog.  You can sort the list of files by name, when last opened, size etc and perform other operations.
