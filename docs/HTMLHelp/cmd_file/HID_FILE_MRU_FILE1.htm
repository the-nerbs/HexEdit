Recently Opened Files (File menu)

 

HexEdit keeps track of the files that you have recently opened. You can display the last 1 to 16 files (the number is set in System page of the Options dialog) near the bottom of the File menu. Just select the file with the mouse or using the numeric shortcut to reopen one of these files.

 

If the file cannot be opened because it no longer exists HexEdit will tell you of this and remove it from the list. Similarly HexEdit will warn you if the file is in use by another program and could not be opened or could only be opened in read only mode.

 

You can see all of your recently opened files using All Recent Files command.
