Error: Illegal Operation

 

HexEdit can crash when you open any file, when templates are turned on.  The text of the message depends on the operating system but usually includes words like "an illegal operation was performed", "abnormal program termination" or even "unhandled exception".

 

This problem occurs under Windows 95, 98 and NT 4 if Internet Explorer 5.0 or later is not installed, or there is a problem with Internet Explorer.  You can confirm that this is the problem by turning off templates (see Display Options page) and checking that opening a file does not crash HexEdit.

 

The solution to this problem is to install or reinstall Internet Explorer 5.0 (or a later version such as 6.0).  You can download the latest version at http://www.microsoft.com/windows/ie/default.asp.
