A toolbar or the status bar is missing

 

You can hide or show any toolbar or the status bar.  If one has accidentally become hidden use the items on the View Menu to show it.  When the status bar or a toolbar is visible it has a tick next to its menu item.

 

Use the View Status Bar Command to hide or show the status bar.  You can also turn the status bar on and off using the context menu that appears when you right-click on it (see Context Menus).

 

Use the View Standard Toolbar, View Edit Bar command, View Navigation Bar command or View Format Bar command for the built-in toolbars or View Custom Toolbar command for any custom toolbars.  You can also show or hide any toolbar using the context menu that appears when you right click on any toolbar (see Context Menus) or in the Customize Toolbars page in the Customize dialog.
