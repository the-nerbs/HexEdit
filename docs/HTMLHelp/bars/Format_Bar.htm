Format Bar

 

 

The Format Bar provides facilities for changing the format of the display.  This includes controls for changing fonts, colors, highlighting etc.  Like the Standard Toolbar it has drop down lists for selecting whether the hex and char areas are display and how characters are displayed in the char area.

 

To hide or display the Format Bar, choose the View Format Bar command from the View menu.

 

To customize this or any toolbar or even create new toolbars use the Toolbar page of the Customize dialog.
