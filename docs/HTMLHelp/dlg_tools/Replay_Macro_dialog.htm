Replay Macro dialog

 

 

Use this dialog to play a macro that has been saved to disk or play the current macro multiple times.  Playing a disk macro does not overwrite or otherwise affect the current (in-memory) macro.

 

Macro Name

Select a disk macro from the list or "Current Macro" to play the current in memory keystroke macro.  The list displays all the macros files in the current macro folder (see the Change Macro Folder button in Macro Options page), except those that begin with an underscore (_).

 

You can created nested macros by invoking this dialog to play a disk macro while recording a macro.  However, you cannot play the "Current Macro" while recording (ie recursive macros are not supported).

Number of Replays

Select the number of time to play the macro.  Note that this option is disabled if you are recording a macro as the number of times a nested macro is played is the value stored when the macro was saved to disk.

Replay Options

Click this button to bring up the Macro Options page of the Options dialog.

Click the Help button to display this information in the on-line help.
