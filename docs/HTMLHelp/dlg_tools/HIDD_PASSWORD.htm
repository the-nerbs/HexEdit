Encryption Password dialog

 

 

Use this dialog to enter the encryption password.  The password is the key used to encrypt data and later decrypt it.  For further details see Encryption.

 

This dialog is invoked by the Encryption Password command or by the Encrypt and Decrypt commands if there is no current password.

 

Password

Enter the password to use for encryption and decryption here  When decrypting you must use the same password with which you encrypted.

Re-enter

If the Mask password check box (see below) is on the you must enter the password again here.  If the value of the Password field (above) does not match this field then the password is not accepted.  This is done to catch typos since you do not have feedback as to the characters typed when masking is used.

Min Length

This sets the minimum password length that you must enter.  For high security make this value at least 8 and always use a good password.  If your requirements are not as stringent you can reduce the minimum password length.

Mask password

When this option is on then when you enter a password you only see asterisks, and you must enter the password twice (in the Password and Re-enter fields).  When this option is off you can see the password as you enter it into the Password field so you don't have to enter it twice.
