General CRC dialog

 

 

This dialog is invoked by the General CRC command.  It allows generating of CRCs of the current selection, placing the result into the calculator.  You can select from several pre-defined standard CRCs or create and save your own.

 

The value for Bits select is very important.  It determines:

  • maximum value for Truncated Polynomial, Initial remainder, and Final XOR fields

  • size of the resultant CRC and hence how good the CRC is at detecting errors

  • minimum number of bits used in the Calculator in order to display the result

Name

This shows the name of the last selected CRC settings.  You can also enter a new name here to save your settings.

Select

Use this list to select one of the pre-defined CRC settings or CRC settings that you have saved..

 

Bits

Select the size of the CRC from the drop-down list.  Valid sizes are currently: 4, 8, 10, 12, 16, and 32.  This value determines the size of the result of the CRC calculation and is also one less than the width of the polynomial expressed below.

Truncated Polynomial

A hexadecimal value that represents the truncated polynomial.  It does not include the top bit.

Initial Remainder

A hexadecimal value that represents the initial value when before the CRC algorithm is begun.  This can be any value but many standard settings use all bits on or off.

Final XOR

This value is XOR'ed with the final output of the CRC calculations..

Reflect Input

This determines the bit order of the inputs.  For example, if using an 8-bit CRC and this is off then bit 7 is the MSB (most significant bit).  If on then bit 0 is the MSB.

Reflect Remainder

This determines the bit order of the final value (before it is XORed with the Final XOR value).

 

Save

Use this button to save a set of CRC settings.  To use it set the CRC parameters (Bits etc) and then enter a name before clicking the Save button.  (The button is not enabled until you enter a unique name.)  The name should then appear in the Select list so you can recall it at any time.

Delete

You can delete any of the settings you have saved using the Save button above.  To do this select the setting from the Select list and click the Delete button. (The button is not enabled unless the Name field contains the name of a CRC settings that you have previously saved.)

OK

Calculates the CRC value and places the result in the Calculator.  The calculator is also set to use Hex mode and if Bits is larger than the Bits setting in the calculator it is increased..
