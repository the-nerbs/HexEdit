Edit Bookmarks command

 

Use this command to work with bookmarks.  It opens the modeless Bookmarks dialog which allows you to add bookmarks, jump to bookmarks, sort bookmarks based on name, file or date added or used etc.

 

As the dialog is modeless you can leave it on the screen to allow you to easily add and jump to bookmarks.
