Copy As Formatted Text (Edit menu)

 

Use this command to copy the selected bytes to the clipboard while converting to text. This command is unavailable if there is no data currently selected. Copying data to the clipboard replaces the contents previously stored there.

 

Note that you can select and copy data even if the file is read only.

 

This command invokes the Copy As Formatted Text Dialog, which provides a large number of options.  For example, you can create a (multi-line) C/C++ string, an array of characters, integers as hex,octal, signed/unsigned decimal, or floating point numbers.  Some aspects of the format of the resulting text, such as the number of columns, case of the hex digits A to F etc, depends on the options in effect for the window at the time the command is executed.

 

This command can be used to extract binary data from a file for use in a C or C++ (or other type of) program.

 

See also the Copy command and Paste command.
