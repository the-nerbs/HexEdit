Select All command (Edit menu)

 

Selects the whole file. You can select all the bytes from the current cursor location to the start of file or to the end of file using Ctrl+Shift+Home or Ctrl+Shift+End.

 

Be careful selecting all of a very large file (many Megabytes in size) and doing some things which may need a lot of memory.  For example, using conversions or encryption on a large file may result in memory exhaustion as HexEdit stores all the converted data in memory.

 

Further there is a 16 MByte clipboard limit. Instead of using the clipboard you can use the Write Selection to File, Append Selection to File and Append Same File commands to build a file from blocks.  See Selecting and the Clipboard for more details.
