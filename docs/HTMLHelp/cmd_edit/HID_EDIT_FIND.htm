Find command

 

Use this command to search the file. This brings up a modeless dialog box that you can leave on the screen and use whenever necessary. It is sometimes quicker and simpler to use the Find Tool on the Edit Bar.

 

The bytes to search for can either be entered in hex or as a string (ASCII, Unicode or EBCDIC). For more information see the Basic Find page of the Find dialog.
