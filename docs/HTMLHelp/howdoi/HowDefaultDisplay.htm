How Do I Change the Default Window Display?

 

When you open a file that is on the recently opened file list HexEdit restores the display the way it was for that file when you last had it opened.  These settings include which area(s) are displayed, the character set, number of columns, font, modifiability, etc as well as the current color scheme.

 

However, when you open a different file or create a new file HexEdit uses the default display settings for showing that file.  You can change the default display settings using the Window Display Options page of the Options dialog.  To do this open the page and adjust any or all of the options in that page as desired, then click the Save As Default button.  If you don't want to change the settings for the active window click the Cancel button else click the OK or Apply button.

 

This will not change the default color scheme.  When a new file is opened the "ASCII default" color scheme is used.  To set the colors for this color scheme use color Options page of the Options dialog.
