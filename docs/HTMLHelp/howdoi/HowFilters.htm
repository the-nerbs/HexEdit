How Do I Only View Graphics Files in the File Open Dialog?

 

By default the File Open and File Save dialogs show all files.  You can change which files are displayed by typing a wildcard specification into the Filename field.  For example typing "*.BMP" will show only .BMP files.

 

Another way is to use the Files of type drop-down list.  The entries in this list allow you to filter the types of files displayed.  For example, if you select Bitmap files (.bmp;.dib) then only .BMP and .DIB files are displayed.

 

You can completely customize the list of filters that are available using the Filters page of the Options dialog.  For example, if you often want to be able to see .BMP and .GIF files you can create a new filter or modify an existing one.
