How Do I Build a Large File?

 

You can use the clipboard to create a large file by copying and pasting.  However, for large operations the normal clipboard formats cause Windows to report memory errors when pasting so HexEdit creates a special clipboard format using temporary files.

 

To more quickly build a large file from pieces taken from other files it may be faster to do this yourself, to avoid the creation of temporary files and the copying of large amounts of data.  To do this you can use of the Append Selection to File and Append Same File commands.

 

You will need to put the bits of the file together one at a time in order.  First use the Write Selection to File command to create a new file or Append Selection to File to append to an existing file.  Thence you can select blocks of text and use the Append Same File command to add further blocks to the end of the same file.

 

Be careful using the Append Same File command.  There is no prompt to confirm that you want to append to the file and no way to undo the operation without opening the file and manually removing the bytes.
