How Do I Find and Replace Bytes?

 

Use the Replace command to invoke the Replace page of the Find dialog to do a find and replace.

 

Searching and replacing using a keystroke macro.

 

In version 2.2 and earlier HexEdit had no find and replace command so this was done with keystroke macros.

 

Use the Tools/Record Macro command to start macro recording (or just use the F7 keyboard shortcut), then search for the bytes you want to replace.  You can use the Edit/Find command or just use the Edit Tool to search for hex bytes or text.

 

When the search is completed the found bytes will be selected.  Just type the value you want to replace them with, then stop macro recording using the Tools/Record Macro command again or F7.

 

You can then repeatedly invoke the Tools/Play Once command (F8) to replace further occurrences.  To do the equivalent of a "Replace All" invoke the Tools/Macro Play command, type a large number such as 10000 into the "Number of Replays" field then hit OK.  The macro will stop replaying when a search encounters the end of file without finding the search bytes.

 

Note that if the macro has undesirable effects you can still easily undo any changes using Edit/Undo Changes.
