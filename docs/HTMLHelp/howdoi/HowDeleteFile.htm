How Do I Delete a File On Disk?

 

HexEdit does not provide any file management facilities; it instead specialises in binary file editing.  However, it is easy to delete, rename etc files in the Windows standard File Open and File Save dialogs.  These are used in the File Open command, File Save command, Write Selection to File, Append Selection to File and Read File commands.

 

 

HexEdit also provides the HexEdit Explorer window which allows you to do most things that you can do in Windows Explorer, like deleting files.

 

To delete a file, just right-click on it and select Delete from the popup menu.
