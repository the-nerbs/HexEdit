How Do I Enter a Floating Point Number?

 

You can easily change IEEE floating point numbers, but you have to know the byte-order and whether it is 32 bit or 64 bit.  If the file is for an Intel platform such as Windows and MSDOS it will be little-endian.  See Floating Point Numbers for more information about floating point number formats.

 

To enter an IEEE floating point number (in decimal), open the Properties dialog (using the Display Properties Command) and select the Floating Point Values Page, then move the cursor to the first byte of the number in the file.  Select the appropriate byte order and format in the dialog.  If you know the value that the file should contain (ie, the value to be replaced) then you can check it now to make sure you have the correct file location and format.

 

Now type the new number into the Value field and press Enter to accept the number you have entered, or press Escape to abort the change.  You can also type in the special values "Inf", "-Inf" and "NaN" (case is ignored) to signify the special IEEE value for +/- Infinity and an invalid number.

 

Note that the Values fields may be disabled if the file is read-only or you are at the end of the file.  If you want to append a number to the file you will have to add some extra bytes to the file (in INS mode) before using the above method to change them.

 

You can also enter IBM floating point values in the same page, but by selecting the relevant format.  The format of these numbers is very different to IEEE and is explained in Floating Point Numbers, and does not support infinity or Nans.  Files containing these types of floating point numbers are from IBM mainframe and mini computers and are usually big-endian.
