How Do I Change a File's Read-Only Attribute?

 

HexEdit does not provide any file management facilities; it instead specialises in binary file editing.  However, it is easy to manage files, including changing attributes, in the Windows standard file open and file save dialogs.  These are used in the File Open command, File Save command, Write Selection to File, Append Selection to File and Read File commands..

 

HexEdit also provides the HexEdit Explorer window which allows you to do most things that you can do in Windows Explorer.

 

To change the read-only attribute of a file on disk, just right-click the file in one of the aforementioned dialogs and select Properties from the popup menu.  See How Do I Delete a File On Disk? to see what this menu looks like.

 

In the General page of the file properties page uncheck the Read-only check-box.  You can also change the archive and hidden attributes here.
