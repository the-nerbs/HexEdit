.topic HIDC_DISPLAY_PREVW
Allows you to choose how to display the bitmap preview.

.topic HIDC_OPEN_SHARED
When the file is opened other programs are not prevented from reading or modifying it. Of course, the underlying file may then change at any time.

.topic HIDC_MACRO_NAME
The file name for the macro.  The ".HEM" extension is added automatically.

.topic HIDC_SUBJECT
Enter a brief description of the bug or subject of the email.

.topic HIDC_CSRC_FLOAT_SIZE
Format of IEEE floating point numbers taken from the file.

.topic HIDC_CSRC_ADDRESS
Include original file address (in C comments) on each line.

.topic HIDC_FILL_CLIPBOARD_SAMPLE
Shows what is currently stored on the clipboard (or the first part of it).  May be hex (for hex data) or characters (for ASCII or Unicode data).

.topic HIDC_FILL_DISK
If this option is checked you are prompted for the disk file name as soon as you click OK and the file is immediately written to disk.  For large files (ie many Mbytes or GBytes) this can save a lot of time as HexEdit does not have to create a temporary file.

.topic HIDC_MOD
Binary operator that calculates the remainder when the first value is divided by the second.  Keyboard shortcut = %.

.topic HIDC_UNARY_REV
Reverses the order of the bits, so that the leftmost bit is swapped with the rightmost bit etc.

.topic HIDC_16BIT
Changes the current number size for entry and calculations to 16 bits (2 bytes).  If the previous size was 64 or 32 bits and the current value does not fit into 16 bits an overflow is signalled.  Keyboard shortcut = F11.

.topic HID_BCG_TOOL_DIR
Specifies the directory in which the program is to be run.

.topic HIDC_BIG_ENDIAN
Byte order of integer/float numbers taken from the file.

.topic HIDC_FILE_PATH
The directory where the file is stored.

.topic HIDC_PRINT_SPACE2
Select this option to use double-line spacing on the printer.

.topic HIDC_SCHEMES
List of available color schemes.  Select a scheme from the list before modifying its colors etc in the controls on the right.

.topic HIDC_MACRODIR
Sets the folder where macros are saved and loaded from.

.topic HIDC_LOADMACRO
Loads a macro file and makes it the current macro.

.topic HIDC_BORDERS
Display the boundaries of disk sectors.  When editing disks/volumes this also shows bad sectors with a different background color.

.topic HIDC_MAX
This says whether the active window is displayed normally or maximized.

.topic HIDC_PACK16
Default to 16-byte alignment.

.topic HIDC_DFFD_EBCDIC
Says that strings and characters use the EBCDIC character set unless otherwise specified.

.topic HIDC_DFFD_EXPR
An expression of any type to evaluate.  May also contain a string with embedded expressions enclosed in braces {} with optional format specifications.  For example, to display a field of flags in binary use something like "flags = {header.flags; bin}".

.topic HIDC_DFFD_IF_REPLACE
Allows replacing the current element with a different type of element.  For example, you can replace a DATA element with a STRUCT.  Note that all attributes and sub-elements of the current element will be lost.

.topic HIDC_DFFD_FORMAT_CHAR
Determines how characters are displayed in the Data column of the tree view, for any character fields that do not have explicit formatting.  You can use C-language printf format specs such as "%c", "%d", %02.2x" etc.

.topic HIDC_FOLDER_HIDDEN
Show or hide "hidden" files.

.topic HIDC_CODE_PAGE
Select the code page that is use to show the character at the left.  Depending on the code page the character shown may use more than one byte starting from the current cursor position.

.topic HIDC_DLG_DOCK
This option allows modeless dialog (Properties, Calculator etc) to dock to the sides of the HexEdit window.  Drag floating dialogs to either side of the main window to dock, OR hold down Ctrl key to prevent docking.

.topic HIDC_DLG_MOVE
When modeless dialog are floating on top of the main window they can sometimes obscure what you are doing  This option means they will always get out of the way of the cursor in the active file's window.

.topic HIDC_FOLDER_LIST_SIZE
How many entries in the folder history (Explorer window) are saved when you exit HexEdit.  For privacy set this size to zero.

.topic HIDC_DISPLAY_TEMPLATE
Allows you to select how to display the template (tree) view, which given the appropriate template (.XML) file, lets you view and edit the data in a structured format.

.topic HIDC_BG_STATS_MD5
MD5 digest of the whole file

.topic HIDC_BMP_WIDTH
Width in pixels of the bitmap in memory
