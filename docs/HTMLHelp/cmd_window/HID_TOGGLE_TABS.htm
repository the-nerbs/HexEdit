Toggle Window Tabs (Window menu)

 

Use this command to turn window tabs on or off. These are a row of tabs that appear at the top (or bottom) of the main HexEdit screen, one for each open window.

 

Window tabs allow you to easily switch to any open window. Right-click any tab to select from a context menu performing commands on the file. Hover the mouse over the tab to se the full path of the file.

 

Turn off tabs to use other window arrangements like Tile Vertically command, Tile Horizontally command, and Cascade command.
