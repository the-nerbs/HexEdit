Options Dialog

 

Options are divided into 3 categories.  Document options are options that can be modified on a file by file basis - if you have no files open then you will not see this section.  Workspace options are global options related to the display and editing of files.  The System section contains all other options.

 

 

System Options

 

The General page contains startup and history options plus options to prevent multiple instances of HexEdit from being run, put “Open with HexEdit” on shell shortcut menus and whether the options you have changed are saved when you exit HexEdit. You can also set the options how you want them and save them immediately with the “Save Now” button.

 

The File Folders page allows you to set the default folders used when you open a file dialog (File Open or File Save As).

 

The Filters page allows you to configure the drop down list of filename filters that is displayed in the File Open and File Save dialogs.

 

The File Backup/Export page allows you to set options for how backups are done when you edit a file and some default setting for exporting data.

 

The Printer General Options and the Printing Decorations pages allows you to change how pages are printed.

 

The Macro Options page has options for how macros are run.  There are also options to save the current macros. load a previously saved macro and change the folder where macros are saved.

 

There are many places that drop-down history lists are available in HexEdit.  The History List Options page allows you to say how much of this history is kept between sessions, as well as other options such as clearing all bookmarks on exit.

 

Workspace Options

 

The Global Options page has global layout options and settings.

T

The Display Options page has options for how all files are displayed.

 

You can set HexEdit to always maximize it's main display when it is opened, and display tabs along the top or bottom of the main window to allow you to quickly flip between files.  You can also set whether hex digits are displayed in uppercase (A-F) or lowercase (a-f) wherever hex numbers are displayed or entered. You can also choose whether you want nice looking addresses displayed – that is, thousands separators in decimal addresses or four digit grouping in hex addresses.  There are also options to change the size of the cursor and to have a 2nd "shadow" cursor in the other (hex/character) area.

 

The Editing Options page has global editing options including turning on background searches, determining whether backup files are created, export options etc.

 

The Background Processing Page allows you to determine when and how background searches and background statistics are done.

 

The Info Tips Page allows complete control over what appears in the "tip" window that appears when the mouse hovers over a byte in the file display.

 

The Template Options page includes global templates options.  It also allows you to control how templates are used for files never before opened in HexEdit and to change the location where templates are kept.

 

 

Document Options

 

The General Options page allows you to restore the default options for the active file.  It also allows you to save all the active document options as the default which determines how new windows are displayed.

 

The Display Options page allows changing all of the display settings of the active window.  Most of these options are available from toolbar buttons but a few (such as setting the column grouping size) are not.

 

The Edit Options page

 

The Color Options page allows you to change any of the colors of the active window's display, including background and the color that address and even different ranges of byte values are shown in.  Moreover, you can create complete color schemes that can be easily selected from the "Color Scheme" drop-list of the "Standard Toolbar".

 

You can set the color scheme used with the active file here, which is why it is in the Document section, but you can also change the colors in any color scheme which may affect all files.  This page appears in the Workspace section if there is no file open.
