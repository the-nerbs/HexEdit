Negate Double Word command (Operations menu)

 

This command changes the sign of the double-word (4 bytes) after the cursor or all double-words in the selection if there is one. This command is not available if the cursor is too close to the end of the file or the number of bytes selected is not divisible by 4.

 

The value(s) are assumed to be in 2’s complement Intel format with the least significant byte first (little-endian). To negate using big-endian format use the Flip Double Word command before and after this command.
