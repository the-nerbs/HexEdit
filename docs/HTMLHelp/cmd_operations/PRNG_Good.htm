Generate Good Random Numbers (Operations menu)

 

This command generates random numbers using an algorithm that will produce statistically good random numbers with a very long cycle.  It combines two algorithms (Mersenne Twist and RANROT) both of which individually have good statistical properties.  (Note, however we have not done comprehensive statistical tests of the numbers generated.)

 

To generate a new sequence of random numbers either place a random value in the calculator and use the Seed Random Number Generator from Calculator command, or just use the Random Seed command.  To generate a reproducible sequence of random numbers load a specific value into the calculator and call the Seed Random Number Generator from Calculator command.
