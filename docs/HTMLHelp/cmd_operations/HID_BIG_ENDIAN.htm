Big Endian (Operations menu)

 

Turns on the big-endian flag for the window.  Note that when you open the same file that this flag will be restored.

 

The big-endian flag affects the byte order used in several operations (assign, negate, increment, addition, subtraction, multiplication, AND, OR, XOR etc).  This flag is also shown in the big-endian checkbox in the Calculator in the top left corner (which is used for byte order when number are read from (or written to) file to (from) the calculator), and the BE/LE status bar pane.
