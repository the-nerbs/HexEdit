Greater (Signed) Byte command (Operations menu)

 

This command compares the byte at the cursor position to the value in the calculator and replaces it if the value in the calculator is larger.  Hence you can use this command to ensure that all values are set to be at least a minimum value.  Note that the comparison is done using signed (2's complement) arithmetic.

 

If there is a selection then it operates on all bytes of the selection.

 

This command is not available if there is no active window or the cursor is positioned at the end of the file.  Further, the number of bits in the calculator must be set to 8 bits, since this command only operates on bytes.

 

Also see the Less Byte Command.
