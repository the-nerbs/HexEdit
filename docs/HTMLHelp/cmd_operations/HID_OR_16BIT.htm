OR Word Command (Operations menu)

 

This command performs a Boolean OR of the current value in the calculator with the word at the cursor position.  If there is a selection then it operates on all words of the selection.

 

This command is not available if an odd number of bytes are selected there is no active window or the cursor is too close to the end of the file.  Further, the number of bits in the calculator must be set to 8 or 16 bits, since this command only operates on 16 bit values.

 

Also see the AND Word command.
