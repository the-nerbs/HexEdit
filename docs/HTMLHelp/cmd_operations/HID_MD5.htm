MD5 command

 

This performs a cryptographic checksum (also called a message digest or one-way hash function) on the currently selected bytes using the MD5 algorithm.  The result is a 128-bit integer which is displayed as hex (as is conventional) in the calculator window. The calculator is switched to hex mode and 128-bits (if currently less).

 

To copy the MD5 value to the clipboard just press Ctrl+C when it is shown.

 

It is (or was once) strongly believed to be computationally infeasible for someone to calculate a set of bytes that would generate a specified MD5 value.  In other words if you know the MD5 value for a set of bytes you can be sure they have not been tampered with by checking that they still generate the same value.  (This contrasts with CRCs which, although excellent for detecting accidental changes to data, are no guarantee that the data has not been deliberately tampered with.)

 

An MD5 value is often used to verify that files downloaded from the Internet have not been modified.  To generate an MD5 value for a file using HexEdit just select the whole file (eg Ctrl+A) before using this command.  Given a choice you should prefer the SHA1 command to MD5. Even better is to use a SHA-2 variation such as SHA 256 or SHA 512.
