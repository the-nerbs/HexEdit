Base 64 Encode command

 

Converts the selected binary data to text using Base64 encoding where every group of six bits is stored as one printable character.  This expands the selection length by 33% because every 6 bytes becomes 8 bytes after encoding.  This uses the common RFC 4648 standard which uses the upper and lower case letters (52 characters) plus 10 digits and the characters plus (+) and forward-slash (/).

 

This form of encoding is similar to but not compatible with uuencode and BinHex. It is also used within MIME.

 

To decode use Base 64 Decode.

 

Base 64 is more efficient than Base 32 Encode increasing the size of the data by only about a third (33%). See Wikipedia for more information.
