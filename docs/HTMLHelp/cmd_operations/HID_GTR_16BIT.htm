Greater (Signed) Word command (Operations menu)

 

This command compares the word at the cursor position to the value in the calculator and replaces it if the value in the calculator is larger.  Hence you can use this command to ensure that all values are set to be at least a minimum value.  Note that the comparison is done using signed (2's complement) arithmetic.

 

If there is a selection then it operates on all words of the selection.

 

This command is not available if an odd number of bytes are selected there is no active window or the cursor is too close to the end of the file.  Further, the number of bits in the calculator must be set to 8 or 16 bits, since this command only operates on 16 bit values.

 

Also see the Less 16 Bit Command.
