Modulus Word command (Operations menu)

 

This command divides the word at the cursor by the value in the calculator and replaces it with the remainder. If there is a selection then it operates on all words of the selection.

 

This command is not available if an odd number of bytes are selected there is no active window or the cursor is too close to the end of the file.  Further, the number of bits in the calculator must be set to 8 or 16 bits, since this command only operates on 16 bit values.

 

Also see the Multiply 16 bit Command and Divide 16 bit Command.
