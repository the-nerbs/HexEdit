Convert EBCDIC to ASCII command

 

This command converts all bytes in the current selection from EBCDIC to ASCII. Any values that cannot be represented in ASCII are converted to a zero byte value.

 

You can also convert from EBCDIC to ASCII by copying to the clipboard (Copy command) while in EBCDIC display mode and then using the Paste As ASCII command. However, this will remove any EBCDIC characters that cannot be represented rather than replacing them with a nul byte.

 

Also see the Convert ASCII to EBCDIC command.  Use the EBCDIC Characters command to toggle the display of characters between EBCDIC and ASCII.  This also affects how characters are copied to the clipboard.
