Subtract Quad Word Command (Operations menu)

 

This command subtracts the current value in the calculator from the quad word at the cursor position.  If there is a selection then it operates on all quad words of the selection.

 

This command is not available if the number of bytes selected is not a multiple of 8, there is no active window or the cursor is too close to the end of the file.

 

Also see the Add Quad Word command and the Subtract From 64 Bit Command.
