16 Bit Checksum (Operations menu)

 

This calculates the checksum of all the words selected and stores the 16-bit result in the calculator.  If the calculator is in 8-bit mode then it is changed to 16-bit mode to hold the result.  If there is no selection or it is not an even number of bytes then the command is not available.

 

This command calculates a checksum by adding all the words, and throwing away any carry.  Sometimes a different "checksum" is used which is just the value added to the others so that all the words add up to zero -- for this type of checksum just negate the value (using the +/- button) in the calculator once you have called this command.

 

A checksum is a simple method of checking that all the bits of the words have not changed.  However, it is fairly easy for compensating errors to occur which generate the same checksum.  A better result is obtained by using a checksum with more bits.  An even better way to check for this sort of error is to use a CRC (see CRC 32 and CRC CCITT).
