Flip Word command (Operations menu)

 

This command reverses the position of the two bytes following the cursor. If there is a selection it reverses the bytes in all words in the selection. This command is not available if there is no active window or there are not two bytes before the end of file or an odd number of bytes are selected.

 

Flipping the order of the bytes in the word allows you to switch between big-endian and little-endian number formats. The order of the bits within each byte is not affected.
