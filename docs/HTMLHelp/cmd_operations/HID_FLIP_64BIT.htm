Flip Quad Word command (Operations menu)

 

This command reverses the order of the eight bytes following the cursor. If there is a selection it reverses the bytes in all quad-words in the selection. This command is not available of there is no active window or there are not eight bytes before the end of file or the number of bytes selected is not divisible by 8.

 

Flipping the order of the bytes in the quad word allows you to switch between big-endian and little-endian number formats. The order of the bits within each byte is not affected.
