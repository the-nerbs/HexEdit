Customize Options Page

 

 

This page has miscellaneous customization options, for toolbars and menus.

 

Show ScreenTips on toolbars

When this option is on a tooltip is displayed for toolbar buttons. This is a short description of the button that appears if you leave the mouse cursor over the button for a second.

Show shortcut keys in ScreenTips

When this option is on, the tooltip displayed for a button also displays the keyboard shortcut (if any) associated with the toolbar button.

Large Icons

This option doubles the size of buttons on all toolbars.

Visualizations

Use this button to show the skins dialog.  This allows you to select a skin which determines the look of HexEdit 's menus and toolbars.  You can even download new skins from the Internet as they become available.

 

Menus show recently used commands first

When menus are first displayed they show only basic items plus items you have recently used.

Show full menus after a short delay

All menu items are shown after a short delay.

Reset my usage data

Resets tracking of used menu items so that menus only show basic items and not items you have recently used.
