Customize Menu Page

 

 

This page allows you to change the main (frame window) menu and the eight popup (context) menus.

 

The “Default” frame window menu is shown when no files are open. The “HexEditPro” frame window menu is shown when there is at least one file open; it contains many more commands than the Default menu.

 

There are three context menus corresponding to different areas of the HexEdit file display: the main hex area, the character area to the right of the hex area (if shown) and the address area to the left of the hex area. In addition, there are context menus for a right-click over the selection or over a highlighted area (see Highlighter) or on a bookmark.  There is also a context menu that is displayed when you right-click on the status bar.  One context menu that is commonly customized is the one that appears when you right-click a window tab.

 

Show Menus for

Select the menu from this list that you want to customize then go to the Commands Page.

Reset

Resets the current menu selected above to its default layout.

Menu animations

This list determines how menus appear on the screen when the menu is opened. The best selection is “None” unless you have a fast machine and video card.

Menu shadows

If this option is selected, menus are drawn with a drop-shadow giving a nicer 3-D appearance.

Select context menu

Select from one of the following to display the context menu in order to customize it.

  • Address area: context menu shown when you right click on the address area

  • Hex area: context menu for the hex area (unless you right click on the selection, a bookmark, or highlight)

  • Char area: context menu for the char area (unless you right click on the selection, a bookmark, or highlight)

  • Status bar: context menu shown when you right click the status bar

  • Selection: context menu shown when you right click the selection

  • Highlight: context menu shown when you right click a highlight

  • Bookmarks: context menu shown when you right click a bookmark

  • Window tabs: shown when you right-click any of the window tabs that correspond to the open files

Select the blank entry (all dashes) to hide the context menus.

Reset

Resets the currently selected context menu to its default layout.
