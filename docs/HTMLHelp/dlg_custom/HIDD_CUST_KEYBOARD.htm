Customize Keyboard Page

 

 

This page allows you customize the keyboard. Any command can be assigned to any shortcut key.

 

When a keystroke is assigned or reassigned the keyboard shortcuts shown to the right of menu items and shown in toolbar tooltips (or “Screentips”) are updated to reflect the new keyboard layout. However, the shortcut keys mentioned in the on-line help only shown the default keyboard shortcuts.

 

You can use the Keyboard Map command on the Help menu to view and print the complete current keyboard layout.

Category

Selecting a category from the list displays all the commands for that category in the Commands list (below it). This is the same list as in the Commands Page.

 

The Macro category allows you to assign any keystroke macro you have saved to disk to be replayed with a shortcut key.

 

The All Commands category lists all commands that are not normally available from menus (only from toolbar buttons or default keyboard shortcuts). The “All Commands” category lists all commands.

Commands

Shows the commands in the currently selected category. Select an item in this list to view or change the keyboard shortcut for the command.

Description

This provides a brief description of the currently selected command (above).

Set Accelerator for

This lists the keyboard tables available. But there is only one – the “Default” table.

Current Keys

When a command is selected in the list to the left, this shows the keyboard shortcut(s) assigned to that command, if any. To remove a shortcut select an entry in this list and click the “Remove” button.

Press New Shortcut Key

When a command is selected you can enter a keyboard shortcut here. Once entered click the Assign button to assign the shortcut to the command. Before you assign it you may want to check if the key is already in use (see below).

 

Keyboard shortcuts consist of most of the keys on the keyboard in combination with one or more “shift” keys (Shift, Ctrl, and Alt). Some keys (such as Ctrl+Alt+Del) cannot be assigned.

 

If the keyboard shortcut is already in use then the command it is assigned to is shown under this control. If it is not in use then “[Unassigned]” is displayed.

Assign

To assign or reassign a keyboard shortcut, select the command (you may need to change the Category first), and type the shortcut key into the “Press New Shortcut Key” field and click the Assign button.

Remove

To remove a keyboard shortcut from a command, select the command (you may need to change the Category first), then select the assigned shortcut in the “Current Keys” list and click the Remove button.

Reset All

Click this button to set the keyboard layout back to the default.
