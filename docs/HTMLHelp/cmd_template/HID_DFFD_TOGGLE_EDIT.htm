Allow Editing (Template menu)

 

This toggles between edit and view modes for the current template.

 

As well as the obvious difference (that you can edit the template when this option is toggled on) there is another important difference.  In edit mode many more template nodes in the tree view are visible which are hidden to a viewer of the template.  Seeing all the nodes allows full control of how the template works.  See Editing templates for details.

 

Note that some templates may be marked as not editable.  In this case you cannot change the template from within HexEdit - you need to modify the XML of the template directly.
