Split Aerial View

 

Displays the aerial view associated with the active file in a split window to the right of the normal display (data view).

 

You can drag the splitter (the dividing bar between the two views) using the mouse to adjust the relative size of the displays.  The next time you open the file in HexEdit the layout is restored.

 

If you have limited screen space and want to see more of the aerial and/or data view (but not at the same time) you should try the Tabbed Aerial View instead.
