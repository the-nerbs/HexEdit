Workspace Display Options Page

 

 

This page allows you to change some global display options that affect all document windows.

 

Document Display Options >>

This buttons jumps to the page for display options of the active file.  This button is disabled if there is no file open.

Display

Large (block) cursor

A more easily visible cursor (or “caret”) is used to show the current edit position in the active window. When off, the cursor is a thin flashing line to the left of the current edit position. When on the cursor changes to be a much wider block covering the character at the edit position. This is useful for people using a small monitor at high resolution or for people with poor eyesight.

 

Show selection length

When on a tip window pops up when selecting bytes using the mouse or the keyboard.  This window shows the length of the selection in hex or decimal depending on how addresses are currently displayed.

 

You can also see this window by holding down the SHIFT key.

 

Divisibility by largest power of 2

When on this shows extra information next to the aforementioned selection length (ie in the tip window).  This is the largest power of 2 (1,2,4,8,16, ...) that exactly divides the length of the selection.  This option is very useful for some operations - for example, some many block encryption algorithms require that the length of the selection is a multiple of 8 bytes.

 

Nice Addresses

Displays addresses in a nicer format.  For hex addresses the means digits are grouped into fours.  For decimal addresses numbers are formatted to local specification (e.g. with thousands separators) as specified in the Window Control Panel.

 

Display "shadow" cursor in Hex/char Area

When you are editing you current edit position is marked with a flashing cursor.  You can edit in the hex or character area but it is handy sometimes to see what the corresponding value in the other (character or hex) area is.  When this option is on the current value in the other area is indicated also.

Ruler

Ruler

Display a ruler at the top of each window.  This display hex and/or decimal offsets depending on whether decimal and/or hex addresses are show (see Document Display Option page).

 

Large ticks every

How often are large tick marks displayed in the ruler instead of small tick marks.

 

Numbers every

How often are numbers shown in the ruler.  Normally the numbers are displayed as a single digit between columns but for the special value of 1 the 2 digits are displayed at the top of each column (but ticks are still displayed between the columns).

 

Indicate cursor position

The current location of the cursor is highlighted in the ruler and the address area.  That is the current column is shown in the ruler and the current row in the address area.

 

Indicate mouse location

As you move the mouse over the active file window the current mouse position is shown in the ruler and the address area.  Turn this option off if you find the continuous movement annoying.

 

Positions are not shown in the ruler unless numbers are shown above every column (i.e. the Numbers every option above is set to 1).

Character Area

When displaying multi-byte characters sets (see Code Page command) some byte positions in the character area do not have a valid characters.  These options determine what is displayed as a place-holder.  For example, for a 3 byte character, the first character position displays the actual character represented by the three bytes, followed by two continuation characters in the following two character positions.

 

Continuation Character

For a multi-byte character the first byte position displays the actual character, and subsequent (continuation bytes) are displayed using this symbol.

 

Invalid Character

Some byte sequences are invalid in most multi-byte character sets.  This option determines the symbol shown for invalid characters.
