Printing General Options Page

 

 

The Printer Options page gives you a great deal of control over how pages are printed.  Not only can you set margins and line-spacing but you can created custom headers and footers with substitutable parameters (see below).

General

Print border

If selected a line is drawn around the page (not including the header and footer).

 

Column headings

If selected headings are drawn at the top of each page showing the offset of the column from the address on the left.  The column numbers are printed in hex if hex addresses are being used, else they are decimal (see Decimal Addresses command).

 

Print mark

If selected the mark is printed as displayed on screen.  On a color printer this is displayed according to the current color scheme.

 

Bookmarks

If selected bookmarks are printed as displayed on screen (as long as bookmarks are not hidden)

 

Highlights

If selected highlights are printed as displayed on screen (as long as highlights are not currently hidden)

 

Search Occurrences

If selected all background search occurrences printed as displayed on screen.

 

Change Tracking

If selected, change tracking marks are printed as displayed on screen.

 

Units

The units of measurement for margins and header/footer positioning.  This will default to local conventions (set in the Windows Control Panel), but you can change it to inches or cm as desired.

Line Spacing

Single Line Spacing

Lines are spaced as shown on screen.

 

1.5 Line Spacing

The distance between printed lines is 50% more than single line spacing.

 

Double Line Spacing

The distance between printed lines is double that of single line spacing.

Margins

Left Margin

Minimum amount of blank space to leave on the left side of the page.

 

Right Margin

Minimum amount of blank space to leave on the right side of the page.

 

Note that the left and right margins may appear larger than specified here, depending on the font in use and the numbers of columns on screen.  HexEdit will print in the same size font as used on screen unless there is insufficient room in which case it will use the largest font size that fits within the left and right margins.

 

Top Margin

The amount of space to leave at the top of each page.  If you have specified a header you must leave enough space for it.

 

Bottom Margin

The amount of space to leave at the bottom of each page.  If you have specified a footer you must leave enough space for it.

From Edge

Header distance from edge

Distance of the page header from the top of the page which allows it to be positioned vertically.  The header must be positioned within the top margin -- if this value is too large then the header will not be printed.

 

Footer distance from edge

Distance of the page footer from the bottom of the page which allows it to be positioned vertically.  The footer must be positioned within the bottom margin -- if this value is too large then it will not be printed.
