System General Options Page

 

 

This page allows you to change some global system settings.

 

Save settings on exit

Selecting this option forces all the current global settings to be saved when HexEdit is closed. Other less obvious things are also saved like the most recent positions of toolbars and modeless dialogs. A simpler way to customize your options is to set them how you want them and then use the Save settings now button (see below).

 

Save settings now

Clicking this button forces all the current options to be saved immediately (even if the Save settings on exit option is not selected).

 

Add “Open with HexEdit” to Shell shortcut menus

This allows any file to be opened with HexEdit using the shell shortcut menus which you see when you right-click on a file in Windows Explorer. The menu item says “Open with HexEdit”. When you click this menu item the file is opened in HexEdit.

 

Note that this option causes problems for some ill-behaved program launchers such as the Office 97 shortcut bar. If you use these then you will have to disable this option in HexEdit.

 

Restore all "Not Shown" Warning and Information Messages

Selecting this makes all "Not Shown" message boxes (where you have selected the checkbox that says either Do not show this message again or Don't ask this question again) to be shown again.

 

Quite often HexEdit will show message boxes containing a warning message or asking a simple question.  Many of these message boxes can appear often and you may become sick of seeing them so they include a checkbox to prevent them from appearing thenceforth.  Once you have turned off the warning or question you will never see it again except by using this button.

 

Only permit one instance of HexEdit to run

If you run HexEdit but it is already running then the new instance does not run, rather the original instance is brought to the front. Any files specified on the command line or via the shell are added to the files already open in the original instance.

 

Restore main window position

The main HexEdit window is restored to its position when you last ran the program. For this option to have any effect the settings must be saved (see the Save settings options above) since it is only effective when the HexEdit starts up.

 

Background Scan for Devices

When on HexEdit scans for hardware devices in a low-priority background thread just after startup.  This allows the "Open Special" dialog to open much more quickly.  Turn this option off to get rid of the annoying floppy access when HexEdit starts.

 

Check for Updates

About once a month when HexEdit starts, it checks the web site to see if a later version is available.

 

Show Splash Screen

Turn this option off if you don't want to see the splash screen when HexEdit starts.

 

Show Tip of the Day

Turn this option off if you don't want to see the "tip of the day" when HexEdit starts.

 

Run AutoExec Macro

If you create a keystroke macro and save it with the name "AutoExec.HEM" then it is run whenever you start HexEdit.  If there is a problem with the macro you can prevent it running using this option.

 

History List Options

Clicking this button conveniently takes you to the History Lists page.  (The History options were on this page in previous versions.)
