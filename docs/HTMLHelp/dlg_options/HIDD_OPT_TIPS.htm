Info Tips Options Page

 

 

It is often very convenient to have relevant info displayed in a popup or "tip" window when you hold the mouse over any byte of the active file.  But what is relevant varies from person to person and changes at different times.  This is why the "info tip" window was invented - it allows you completely customize a list of tips and turn them on and of at any time.

 

The tip window displays two columns - on the left is a name or description and on the right is the result of an expression using the current byte(s) being hovered over by the mouse.  The expression and how it is displayed is completely flexible.  The result of the expression can include any of the predefined identifiers listed in the table below.  These usually use the value of the byte being hovered over, or for multi-byte values the byte being hovered over and following bytes.

For example, to display the distance from the "mouse hover point" to the mark use the expression "mark - address" and a format of "hex" or "dec".  To display whether a particular bit in a byte is on you can test the bit using the & operator (eg "(byte & 0x02) != 0") and use a format of "OFF" which displays ON if the bit is 1 or OFF if it is 0.

 

List

The list contains 4 columns.  The first column enables the individual tips. Any tip that is not enabled is not shown in the popup window.  The other columns are described below.

 

    Name

This description helps you remember what the tip is for.  It is displayed in the left column in the popup tip window.

 

    Expression

Evaluated to produce a value displayed in the right column in the popup tip window.  The expression can use any of the identifiers listed in Table 1 above.

 

    Format

This string determines how the result of the above expression is formatted before being displayed.  Please refer to Table 2.

 

Transparency

Drag the slider to adjust the transparency of the info tip popup window.  All the way to the left is fully transparent; all the way to the right is fully opaque.  This is useful if you have a large number of tips but you do not want to obscure what is underneath.  To make the window completely visible (ie opaque) just hold the mouse over it.
