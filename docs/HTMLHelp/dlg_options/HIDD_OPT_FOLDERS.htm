File Folders Options Page

 

 

This page allows you to set the default folders (otherwise known as directories) which are used when opening or saving a file.

Default Open Folder

The File Open dialog is used whenever you use the File Open command.  You may choose one of these options:

 

Folder of Active Document - the File Open dialog opens to the folder where the currently active file is from - if no file is open then the Specified folder (below) is used

Folder last used in Open Dialog - the File Open dialog opens to the same folder as when last used

Folder last used in Open/Save Dialog - the File Open dialog opens to the same folder which was used for the last File Open or File Save

Specified - the File Open dialog always uses this folder

Default Save As Folder

The Save As dialog is used whenever you use the File Save As command or you save a file (eg, using File Save command) which has never been saved before.  You may choose one of these options:

 

Folder of Active Document - the Save As dialog opens to the folder where the currently active file is from - if the active file is new (ie, never saved) then Specified folder (below) is used

Folder last used in Open Dialog - the Save As dialog opens to the same folder as when last used

Folder last used in Open/Save Dialog - the Save As dialog opens to the same folder which was used for the last File Open or File Save

Specified - the Save As dialog always defaults to this folder
