New Color Scheme Dialog

 

 

This dialog is invoked from the New Color Scheme button in the Color Options page of the Options dialog.  You can add as many custom color schemes to HexEdit as you like.

 

Note that you cannot add (or delete) standard color schemes.  There are always four standard color schemes.  (However, you can change all the colors and ranges in the standard color schemes as usual.)

 

Name for new scheme

This is the name for the new custom color scheme which will appear in the Schemes list in the Color page of the Options dialog and in the Color Scheme Tool.  You must enter a unique name, ie, a name different from all the existing (standard and custom) color schemes.

 

If the scheme is given a name that starts with an underscore followed by a file extension, then when a new file of that type is opened or created the scheme is automatically used for that file.  For example, a scheme named "_EXE" will be used when you open any .EXE file such as PROGRAM.EXE.

 

Copy settings from

You can copy settings from an existing (standard or custom) color scheme to save time setting all the colors and ranges.
