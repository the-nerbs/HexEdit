Document Editing Options Page

 

 

This page allows you to change options to do with editing of the file open in the active window.

 

Global Editing Options

This button jumps to the page for global (not document specific) editing options.

 

Modifiability - Read only

The active window is read only. The file cannot be modified through this window unless read-only mode is turned off. Note that if the file was opened read only then you cannot turn off read-only mode.

 

Modifiability - Read/Write

The opposite of read-only mode. The file can be modified through this window.

 

Mode - Overtype

Any data added to the file overwrites existing data. Backspace does not delete bytes but changes them to null bytes (in the hex area) or space (in the character area).

 

For most types of binary files this is the safe option, since nothing can be inserted into or deleted from the file. Many types of binary files (such as .EXE files) rely on the position of bytes within the file not changing. Further, if you insert or delete bytes from a large file the disk file cannot be modified in place and must be copied to a temporary file before the original is deleted and it is renamed. This can be slow and increases the chances of running out of disk space.

 

Mode - Insert

The opposite of overtype mode. Bytes can be inserted into or deleted from the file.

 

Use Big-endian byte order for operations

This affects the byte order used for commands on the Operations menu.  It is also the same as the Big-endian checkbox in the calculator, which is used with certain calculator buttons which read and write to the active file.

 

Scroll Zone

Says how close to the top or bottom of the window the cursor must be before scrolling is performed.  If the value is zero you can move right to the edge.  If the value is large (eg, 999) then the current cursor position will remain in the (vertical) middle of the window and using the up and down arrow keys just causes the window to scroll.  In Stacked mode (see Show Areas above) the scroll zone is automatically increased to 2, so you can see all of the information about the current byte.

 

Show inserted bytes

This is the first of the four change tracking options.  This particular options highlights which bytes have been inserted in the file by using a light shade of the change tracking background colour.  This is red by default in most color schemes but can be changed by modifying the active color scheme.

 

Show modified bytes

This shows which bytes have been changed by underlining them in red (in the default color scheme).  Note that this reflects the current changes to the file when compared to what is on disk - for example, if you delete 10 bytes then insert 10 bytes (in INS mode) this just shows that 10 bytes have been modified, the same as if you had just overwritten the bytes (in OVR mode).

 

Show deleted bytes

This shows where bytes have been deleted using a small red rectangle (in the default color scheme).  In the hex area this is a large rectangle containing a single digit count of the number of bytes that have been deleted or an asterisk (*) - see below.  In the hex area, and in stacked display mode this is just a thin red line between the bytes.

 

Deletion count

This is a change tracking option that is a only available if the "show deleted bytes" option is on.  It only applies to the hex area where there is enough room between bytes to indicate how many bytes have been deleted.  If the option is on it displays a single digit byte count (1 to 9) between the bytes where the deleted bytes were, or an asterisk if more than 9 bytes were deleted.  If the option is off it always displays an asterisk.

 

Show bookmarks

Turning this option off hides the bookmarks without actually deleting them.  When on, bookmarks are shown using a background color taken from the current color scheme (often a light blue).

 

Show highlights

Turning this option off hides all highlights in the active file without actually deleting them.  When on highlights are shown with a background color taken from the current color scheme (often yellow).
