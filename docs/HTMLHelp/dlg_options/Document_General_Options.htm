Document General Options Page

 

 

This and the following pages set options for the currently active document.  If no files are opened then these page are not displayed, except for the Color Schemes page (which can be used to edit Color Schemes but not to set the active one).

 

Use Default Settings

This button resets all the options to the current default settings (see Save As Default below).  Of course, they do not take effect until the OK or Apply button is clicked.

 

Save Current As Default

This button saves the current options as the default settings.  The defaults settings are used when a new file is created or if you open a file that you have not opened before or one that is not on the recently opened file list on the File menu. If you open a file on the recently opened list HexEdit restores the settings in effect when that file was last used.

 

Template View Display

This drop down has three options that determine how the template view is shown.

 

No Template View

The template view is not shown.

 

Split Window

The template is displayed to the left of the normal (hex) view in a split window.  This allows you to see the template view and the data view at the same time and even adjust their relative sizes by dragging the splitter.

 

Tab Window

The template tree view is displayed in a separate tabbed window.  You can only see one of the normal view or the tree view at a time but can easily switch between them by clicking on the appropriate tab.

This drop down has three options that determine how the aerial view is shown.

The aerial view is displayed to the right of the normal view in a split window.  This allows you to see the aerial view and the data (hex) view at the same time and even adjust their relative sizes by dragging the splitter.

 

Tab Window
