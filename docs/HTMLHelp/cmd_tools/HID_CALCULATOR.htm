Calculator command (Operations menu)

 

This command invokes HexEdit’s built-in calculator. You can use the calculator to perform calculations on 8, 16, 32 or 64 bit integer values in hex, decimal, octal or binary. The calculator has many facilities for manipulating the active file which are very powerful especially when combined with keystroke macros.  The calculator is also used in the Go To command.

 

See the Calculator for more information on performing calculations. See Macros for more information on using keystroke macros.
