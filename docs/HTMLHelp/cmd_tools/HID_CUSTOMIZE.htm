Customize command

 

This displays the Customize dialog, which allows you to configure the user-interface of HexEdit.

 

This dialog has the following pages:

Commands Page shows the commands that can be used on toolbars and menus

Toolbars Page allows you to create and customize toolbars

Tools Page allows you to add user-defined tools

Keyboard Page allows you to customize the keyboard

Menu Page allows you to customize the main menu and various context menus

Mouse Page allows you to customize double-click events in various parts of the display

Options Page has miscellaneous customization options
