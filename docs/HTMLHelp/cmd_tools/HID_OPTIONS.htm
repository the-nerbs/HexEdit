Options command

 

This displays the Options dialog, which allows you to change the behaviour of HexEdit.

 

This dialog has the following pages:

General Options page

File Folders page

File Type Filters page

File Backup/Export page

Printer General Options page

Printing Decorations page

Macro Options page

History List Options page

Workspace Global Options page

Workspace Display Options page

Workspace Editing Options page

Background Processing Page

Info Tips Page

Template Options page

Document General Options page

Document Display Options page

Document Edit Options page

Color Options page
