Toolbar Tools

 

Several tools are available on the default toolbars that need further explanation.  These tools consists of text and a drop-down list.  They can be placed on any existing or new toolbar.  They may also be resized by clicking on them when the Customize dialog is open and resize using the resize tool at the right end.

 

Click on the tool pictures below to get more information.

 

Jump Tools

 

      

 

The decimal and hex address tools allow you to enter and address and press Enter to jump to the address.  You can also select a previously entered address from the drop-down history list.  Version 2.5 adds the ability to enter an expression and bookmarks into these tools (see Jumping to an Address in Moving Around).

 

As you move the cursor in the file the tools are also updated to reflect the current address.

 

When the toolbar is docked vertically the tool is replaced by a button that invokes the Calculator in decimal or hex mode.  You can enter an address here and click the "GO" button to jump to an address.

 

Find Tool

 

 

The find tool allows you to search for a string (case-sensitive or case-insensitive) or search for hex bytes.  To perform a case-sensitive string search type an equals sign (=) followed by the string and press Enter; for a case-insensitive search use a tilde (~).  To perform a hex search just type the hex digits and press Enter.

 

This tool also has a drop-down list of previous searches.  This history list is preserved between invocations of HexEdit .

 

When the toolbar is docked vertically the tool is replaced by a button that invokes the Find dialog (in the Basic page).

 

Color Scheme Tool

 

 

The color scheme tool shows the current color scheme selected.  You can choose a new color scheme by selecting from the drop-down list.

 

When the toolbar is docked vertically the tool is replaced by a button that invokes the Color Options page of the Options dialog.  You can select a scheme from the list here also.

 

Bookmarks Tool

 

 

The bookmark tool lets you quickly jump to any bookmark in the active file.  As you move the cursor in the file it is updated to reflect the nearest bookmark above in the file.

 

When the toolbar is docked vertically this changes to a button that invokes the Bookmarks dialog

 

Font and Font Size Tools

 

   

 

The font and font size tools allow you to quickly select the font that is used in the active window.

 

When docked vertically the tools are displayed as buttons the invoke the Choose Font Dialog.
