Ruler

 

 

 

The ruler is shown at the top of each window.  It is mainly a convenience that saves you having to manually count columns in the display but it also provides ways to quickly change how columns are shown.  You can toggle the display of the ruler using View Ruler Command.

 

Hex and Decimal Rulers

 

There are actually two independent rulers - one shows hexadecimal offsets and the other shows decimal.  Which of the hex and/or decimal rulers is shown depends upon how addresses are displayed in the window.  To toggle between hex and decimal addresses use the Decimal Addresses command.

 

You can even see both rulers at the same time.  If both hex addresses and decimal addresses (or decimal line numbers) are shown then both rulers are visible.  Further if the "Addresses start at 1" option is on then all ruler offsets start at one not zero.  See Document Display Options page.

 

Format

 

You can control how often ticks and numbers are displayed independently for the hex and decimal rulers.  When numbers are displayed for every column they are displayed above the column otherwise the numbers are displayed between the columns.  These settings are also independent of column grouping in effect for the window.

 

See Workspace Display Options for the ruler settings.

 

Column Indicators

 

As you move the cursor around in the file the current cursor column is reflected in the ruler (and the row is reflected in the address area).  For example in the above screen capture the cursor is in column 7.  Similarly the current mouse position can be shown whenever the mouse is over the window.

 

If you find the continuous changes in the ruler and address area annoying, these indicators can be turned off in Workspace Display Options.

 

Handles

 

At the bottom edge of the ruler are three handles for the current offset, grouping and number of columns.  See Document Display page for more information on these settings.

 

If you hold the mouse over the handle it will show the current setting.  If you click and drag the handle you can adjust the setting.

 

If Auto Fit mode is on the row size handle is red, but you can still drag it to change the setting.  Double-click the handle to turn on Autofit.
