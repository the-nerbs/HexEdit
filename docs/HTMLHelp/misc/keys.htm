Keyboard Shortcuts

 

The keyboard can be completely customized using the Keyboard page of the Customize dialog.  The default set of keys is intended to suit casual users, who are used to other Windows programs and hex editors.

 

The first consideration in the default keymap was compatibility with commands that are provided in other Windows editors and word-processors – Windows standard keys as prescribed by Microsoft as well as de facto standard keys. We also borrowed keys from MS Word, FED (a popular MSDOS binary file editor), Brief (a popular MSDOS programmers editor), and some de facto UNIX standards (like Ctrl+L to redraw the screen).

 

The tables below show the default key mapping.  If you have customized the keyboard you can view the current key mapping using the Key Map command.
