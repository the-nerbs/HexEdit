Window Tabs

 

The tabs at the top (or bottom) of the main HexEdit window show a tab for each open window.  They make it very easy to switch between open files..

 

Tab Display

 

You can turn off the display of tabs in Workspace Global Options page or by using Toggle Window Tabs.  There are also options to have them displayed at the bottom of the main window and with icons.  You can also re-order the tabs at any time by dragging them.

 

The text in the tab contains the file name.  It is followed by an asterisk (*) if there are unsaved changes.  If more than one window is open on the same file the tab text contains a colon (:) followed by the window number.

 

Hold the mouse over the tab to see the full path name in a "tip" window.

 

Context Menu

 

Right-clicking on a tab brings up a popup or context menu of commands relevant to the file such as File Close command, File Save command, File Print Preview command, Window Tabs at Bottom command and Show Tab Icons command.

 

You can customize the context menu using the Customize Menu page.  Select the "Window Tabs" context menu and drag commands to the menu as you would to customize toolbars or the main menu - see Customization for details.
