New Compare dialog

 

 

Compare with Self

Use this option to compare a file to itself - to see how it changes over time.

 

This option is only available if the file is opened "shareable".  If the file is not shareable then the file cannot be modified (except within HexEdit).

Compare with File:

Enter a file name to compare with.

Browse...

Use the Browse buttons to select a file..

 

Split Window

Open a view on the compared file in a split window.

Tabbed Window

Open a view on the compared file in a tabbed window.

 

Detect Insertions and Deletions

If selected then when a difference is found HexEdit tries to resynchronize the comparison by searching forward in both files for matching bytes. That is conversion from one file to the other would involve insertions and/or deletions (not just replacements).

Minimum Match Length

When searching forward for matching bytes there must be at least seven consecutive matching bytes, but you can use a larger value (up to 64).

 

Auto Sync

The current cursor position or selection is synchronized between the original file and the file being compared with. That is, when you move the cursor in one window it moves to the same location in the other. (This allows for insertions and deletions.)  Similarly  if you make a selection in one file the corresponding bytes in the other are also selected.

 

If insertions/deletions are allowed then the selections may not be the same length.

Auto Scroll

As you scroll one window the other window moves correspondingly.

 

Compare

Click the Compare button to start the comparison.
