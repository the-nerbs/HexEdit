Keep Times command

 

This command toggles the "keep times" option.  When the option is on file times (modification time etc) as stored by the filesystem are preserved when you edit a file.

 

Normally when you view a file it's time of last access is updated to the current time.  When you change a file the modification time is also updated.  Using this command the times that were in effect when the file was opened are restored when you save the file.

 

This can be used so that other software or people are unaware that the file has been changed.  For example, this could be used for changing comments in source files without causing unnecessary rebuilds due to the makefile dependencies (or other build dependency system).
