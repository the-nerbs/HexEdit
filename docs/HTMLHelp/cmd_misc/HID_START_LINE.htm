Left Side command

 

Use this command to move the current column to the left side of the display. Doing this does not change the file in any way, but just changes the columns in which the bytes of the file are displayed. This is useful for analysing bytes in a file – for example, you might like to use this command when you encounter the start of a new record or structure.

 

This command simply changes the value of the window “offset”. See the Window page for more information.

 

Shortcuts

Keys: Alt+Enter
