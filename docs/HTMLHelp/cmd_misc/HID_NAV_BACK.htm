Navigate Backward command

 

Move back to a previous location (or navigation point) in the current editing session.  If the previous navigation point was in a file that is now closed the file is reopened.

 

Navigation points are remembered while editing/viewing files as you move around, or switch between different files.  They are kept in a strict chronological order.  This allows you to easily go back to what you were doing in another place in the file or in another file without having to use undo (which will undo file changes and does not work between files).
