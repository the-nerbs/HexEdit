Endianness Toggle

 

This command allows you to change between big-endian and little-endian mode.  The sixth pane from the right indicates the current status - it displays LE for little-endian and BE for big-endian mode.  This flag is also shown in the big-endian checkbox in the Calculator in the top left corner.

 

This mode affects some operations on the active file, but does not change the display at all.  Note that this is a "per file" flag and when you re-open the same file that this flag will be restored.

 

It affects most commands in the "Operations" menu (where the endianness of the operation is important).  This includes operations which use the current calculator value including binary operations and assignment.  It is also is used for byte order when numbers are read from (or written to) file to (from) the calculator using the file access buttons.

 

Note that byte order is relevant even when you might not think so.  For example, if performing a 16 Bit AND operation on the bytes 12 34 and the calculator contained 00FF (hex) the result would be 12 00 in little-endian mode or 00 34 in big-endian mode.

 

Shortcuts

Mouse: Double click the LE/BE indicator on the status bar
