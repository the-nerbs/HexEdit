Track Changes Toggle

 

Toggles change tracking.  When change tracking is on all differences between the current file and the file on disk (ie, as it was when opened or last saved) are highlighted.  The color used to highlight the changes is taken from the color scheme in use in the active file - use the color Options page to change the Change Tracking color or to select a different color scheme.

 

Changes are shown in this way:

  • Replacements are underlined.

  • Insertions are shown with a highlighted background

  • Deletions have a mark at the byte after the deletion.

    • If less than 10 bytes are deleted the number of deleted bytes is displayed in between the bytes in hex view.

    • If more than 10 bytes have been an asterisk is displayed.

Note that you can only insert or delete bytes if you are in insert mode (see Insert Mode Toggle).  And you can't even replace bytes in read-only mode (see Allow Changes Toggle).

 

Also note that the changes displayed do not reflect the actual editing operations performed only the differences between the current file and what is saved on disk.  For example if you delete 5 bytes then insert 5 bytes at the same location this will be shown as 5 bytes having been replaced.
