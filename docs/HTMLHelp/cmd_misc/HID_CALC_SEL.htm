Calculate Using Selection

 

This commands invokes the calculator loading the currently selected value into the calculator.  Depending on the length of the selection the calculator is automatically set to 8, 16, 32, or 64-bit mode.  If more than 8 bytes are selected then this command is not available.  Note that the current state of the calculator "Big-endian (file access)" check box determines the byte order of the loaded bytes.

 

This command could be used to jump to a file address stored in the file.  Just select the address, whether 32-bit or 64-bit and invoke this command the click the "GO" button in the calculator to jump to the address.

 

Shortcuts

Mouse: Right click the selection and select from the selection context menu.
