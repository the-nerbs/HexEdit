Counts Page

 

 

This page shows a graph of the byte counts for the active file.  It is only displayed if the Statistics option in selected in the Background Processing Page of the Options dialog.

  • The bars of the graph correspond to the number of occurrences of each byte value in the file

  • The bars are shown according to the range colors in the current color scheme

  • Hold the mouse over a bar to the byte value and count for any of the 256 bars

  • You can resize the window to make the graph bigger
