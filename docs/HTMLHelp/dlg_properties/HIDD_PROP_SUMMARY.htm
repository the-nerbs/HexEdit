Summary Page

 

 

This page shows, and allows you to add, several text fields to be associated with the file.  It also shows the amount of time that you have had the file open plus file size information.

 

Category

Enter any value into the category field to allow you to better manage editing of large groups of files.  For example, files can be sorted in the Recently Opened Files Dialog by category.

Category Selection List

The arrow next to the category field displays a list of existing categories to select from.  Using this mechanism avoids having slight variations on the same category name.

Keywords

Enter any number of keywords (separated by spaces and/or commas).  This allows you to easily find files by searching on keywords.

Comments

Enter any comments regarding the purpose of the file or why you were looking at it or editing it, etc.

 

View time

Amount of time that you have had the file open in HexEdit in read-only mode..

Edit time

Amount of time that you have had the file open in HexEdit while not in read-only mode (even if you made no changes to the file).
