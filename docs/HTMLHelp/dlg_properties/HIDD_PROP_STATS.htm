Checksum (Statistics)  Page

 

 

This page shows background calculations if the Statistics option in selected in the Background Processing Page of the Options dialog.

 

CRC32

Calculates a CRC (using standard CRC32 settings) of the whole file.  This is the same CRC used on files stored in ZIP format and many other places.

 

MD5

Calculates the MD5 message digest of the whole file.  This format is the most common value currently used to generate message digests and posted on the Internet.  However, it is deprecated in favour of SHA1 or better still SHA 2

 

SHA1

Calculates the SHA1 message digest of the whole file.  Currently the best commonly available message digest.  Message digests are most commonly used to verify that files downloaded from public places have not been tampered with.

 

SHA 256

Calculates the 256-bit SHA2 message digest of the whole file. This is considered a better algorithm than SHA1 but is not widely supported yet.

 

SHA 512

Calculates the 512-bit SHA2 message digest of the whole file. This uses the same algorithm as SHA 256 but is simply better because there are more bits.

 

You may need to widen the Properties window to see all 64 hex digits of SHA 512.
