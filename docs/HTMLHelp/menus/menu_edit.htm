Edit menu commands

 

The Edit menu offers the following commands:

 

Undo

Reverses the last operation.

Undo Changes

Reverses the last operation that changed the file.

Cut

Deletes data from the file and moves it to the clipboard.

Copy

Copies data from the file to the clipboard.

Paste

Pastes data from the clipboard into the file.

CopyAs Hex Text

Copies bytes to the clipboard converting to hex digits.

CopyAs C Source

Copies bytes to the clipboard converting to C/C++ source code (comma separated values).

PasteAs ASCII

Pastes from the clipboard ignoring non-text ASCII bytes and EBCDIC mode.

PasteAs Unicode

Pastes from the clipboard as Unicode characters.

PasteAs EBCDIC

Pastes from the clipboard as EBCDIC characters.

Select All

Selects the entire file.

Write Selection

Writes the current selection to a file.

Append Selection

Appends the current selection to a file.

Append Same File

Appends the selection to the same file as the last write or append.

Read File

Inserts a file at the current cursor position.

Insert Block command

Inserts a block of bytes with various fill options.

Highlight

Highlight the selected text or toggle highlight mode

Mark Position

Sets the mark to the current cursor location.

Go To Mark

Returns the cursor to the last marked location.

Extend to Mark

Extends the selection to the current position of the mark.

Swap Cursor and Mark

Swaps the positions of the cursor and the mark.

Edit Bookmarks

Invokes the bookmarks dialog so you can add a new bookmark or jump to an existing one.

Find

Searches for hex digits, or a string.

Replace

Search and replace hex digits, or a string.

Go To

Invokes the calculator to jump to a specified address
