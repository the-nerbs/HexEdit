Window menu commands

 

The Window menu offers the following commands:

 

New Window

Creates a new window that views the same file.

Next Window

Activates the next non-minimised window in a cyclic list.

Toggle Window Tabs

Toggles display of window tabs at top (or bottom) of the main display

Cascade

Arranges windows in an overlapped fashion.

Tile Horizontally

Arranges windows as non-overlapped, stacked tiles.

Tile Vertically

Arranges windows as non-overlapped, side by side tiles.

Arrange Icons

Arranges icons of closed windows.

Window 1, 2, ...

Goes to the specified window.

Windows

Manages the currently open windows.
