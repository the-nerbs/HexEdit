Compare menu commands

 

The Compare menu offers commands for comparing a file against another file:

 

New Comparison

Start a new file comparison

Hide Compare View

Close the compare view

Split Window

Show the compare view in a split window

Tabbed Window

Show the compare view in a tabbed window

First Difference

Jumps to the first difference in both files

Previous Difference

Jumps to the previous difference

Next Difference

Jumps to the next difference

Last Difference

Jumps to the last difference

Auto Scroll

Keeps scrolling synchronized between the windows

Auto Sync

Keeps the cursor position (and selection) synchronized
