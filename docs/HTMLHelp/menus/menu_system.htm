System menu

 

The System menu appears when you click the icon in the very top left of a window or if the window is maximised the icon to the left of the File menu.  It contains the following commands for controlling the window:

 

Restore command

If the window is minimized or maximized it restores it to its normal position

Move command

Allows the window to be moved with the cursor keys or mouse

Size command

Allows the window to be resized with the cursor keys

Minimize command

Iconizes the window and places it near the bottom of the main window

Maximize command

Forces the window to use all available space in the main window

Close command

Closes the window.  If this is the last window associated with a file the file is closed.

Previous Window

Moves to the previous window in the list of windows.

Next Window

Moves to the next window in the list of windows.
