Tools menu commands

 

The Tools menu offers the following commands:

 

Find Hex

Searches for hex digits.

Compare Windows

Compares the values in two windows.

Calculator

Displays the built-in calculator.

User-defined Tools

Invoke one of the user-defined tools

Customize

customize user-defined tools, keyboard, toolbars, menus etc

Options

Invokes the Options dialog.

Record Macro

Starts or stops recording of a keystroke macro.

Quick Play

Plays the last recorded keystroke macro once.

Macro Message

Record instructions for display during macro playback.

Play

Plays a number of times, or play a disk macro.
