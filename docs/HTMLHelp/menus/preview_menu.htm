Preview menu commands

 

The Preview menu offers commands to show and manipulate the preview window:

 

Hide Preview View

Close the preview window

Split Window

Show the preview in a split window

Tabbed Window

Show the preview in a tabbed window

Zoom Actual

Set zoom where one bitmap pixel = one display pixel

Zoom Fit

Zoom the bitmap to just fit horizontally and/or vertically

Zoom In

Increase magnification

Zoom Out

Decrease magnification

White Background

Use a white background for transparent and semi-transparent pixels

Black Background

Use a black background for transparent and semi-transparent pixels

Check Background

Use a checkerboard background to best see partially transparent pixels

System Default Background

Use the default window background color (typically light gray)
