View menu commands

 

The View menu offers the following commands:

 

Toolbars – Standard Toolbar

Shows or hides the new Standard Toolbar.

Toolbars – Edit Bar

Shows or hides the Edit Bar.

Toolbars -- Format Bar

Shows or hides the Format Bar.

Toolbars -- Navigation Bar

Shows or hides the Navigation Bar.

Toolbars – Custom toolbar

Shows or hides custom toolbars.

Toolbars -  Calculator

Shows or hides the Calculator

Toolbars - Bookmarks

Shows or hides the Bookmarks dialog

Toolbars - Find Dialog

Shows or hides the Find dialog

Toolbars - Properties

Shows or hides the Properties dialog

Toolbars - Explorer

Shows or hides the Explorer window

Toolbars – Customize

customize toolbars, etc

Status Bar

Shows or hides the Status Bar.

Ruler

Shows or hides the ruler

Application Look

Sub-menu of options to change the overall look

Autofit

Toggles the mode in which the columns are adjusted to fit the window.

Font

Invokes the standard Font Selection dialog.

Decimal Addresses

Toggles between hex and decimal addresses.

Show Hex Area

Shows the hex area (hides the character area)

Show Char Area

Show the character area (hides the hex area)

Show Both Areas

Shows both the hex and character areas

Display Stacked

Shows hex and characters in stacked mode

Char Set ASCII

Display characters using the ASCII character set

Char Set ANSI

Displays characters in ASCII with ANSI extensions

Char Set IBM/OEM

Displays characters in ASCII with IBM.OEM graphics characters

Char Set EBCDIC

Display characters in the EBCDIC character set

Char Set Last Code Page

Displays characters using the last code page used or the default code page

Char Set Code Pages

Sub-menu of options to show characters using common code pages

More (Code Pages)...

Show the Document Display page to allow selection of any code page

Control Chars None

Displays ASCII control characters as a dot in the character area

Control Chars Alpha

Displays ASCII control chars as there corresponding alphabetic char

Control Chars C/C++

Displays ASCII control chars as the C/C++ escape sequence char

Reset To Defaults

Reset options of window to current defaults.

Properties

Invokes the Properties dialog.
