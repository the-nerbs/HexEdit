Installing and Removing HexEdit

 

HexEdit has been tested on Windows 7, 8 and 10. You may have difficulty running under Windows XP or earlier.

 

Installation

  • Before you install HexEdit 5.0 you need to obtain the installation files. The latest release may be obtained from the releases page.

  • When you double-click the hexed5_0.exe the Microsoft Windows Installer (MSI) program is run. Due to changes in how some runtime components are distributed, HexEdit's installer now runs in two parts. It will first run an installation for the Microsoft Visual C++ Redistributable, and then will run the normal HexEdit installation package.

  • Follow the instructions to install HexEdit

  • If you intend to run HexEdit from a console command line then it needs to be installed in your PATH. You can do this by adding the directory where you installed HexEdit to your PATH.

  • If HexEdit is misbehaving or won't run you can try the repair option on the Help menu or the /clean command line option or try uninstalling then re-installing

Removing HexEdit

  • In Control Panel double click the “Programs and Settings” icon.

  • Click HexEdit in the list of files that may be removed and click the Remove button.

  • Click “Yes” if you really want to remove HexEdit . The removal procedure is displayed.

  • You are given the option of retaining or removing your settings.  It is better to retain your settings if you might use HexEdit again.

  • After HexEdit has been removed just click the OK button.

Note: Some things may not be completely removed until you shutdown and restart the system. For example, you may still see HexEdit shown in the Start Menu until you reboot.
