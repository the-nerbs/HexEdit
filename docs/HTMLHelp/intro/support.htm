Support

 

We’d like to hear from you if you want to report a bug, request an enhancement, or just tell us you like the program.

 

If you have a question about using HexEdit please check the extensive help first. You can contact us from our GitHub page, where you can create an issue to submit and bug report or request a new feature, or start a discussion to ask a question.

 

GitHub

 

It is very easy to get in contact with us if you are connected to the internet. Visit the repository on GitHub at https://github.com/the-nerbs/HexEdit. From there, you can submit an issue to report a bug or request a new feature. You can also start a discussion.

 

Important notes:

If you have found a bug include the following:

 

1. A brief description of the problem.

2. Describe the problem as fully as possible.

3. Try to remember exactly what you did and what happened.

4. List the steps required to reproduce the problem if you can reproduce it.

5. Include the exact text of any error messages from HexEdit or the system.

6. Include the operating system version and the version of HexEdit you are using.

7. Try to think of anything peculiar about your configuration that may be relevant to the problem.
