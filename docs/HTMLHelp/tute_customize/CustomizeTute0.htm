Customization Tutorial

 

         

Welcome to the HexEdit Customization Tutorial.

 

 

What is Customization?

 

Customization allows you to configure HexEdit to the way you want to work, or the way you are used to working with other similar software.  HexEdit users who make use of the customization features greatly appreciate how much more enjoyable using the software can become.  Many customization features are great time savers too.

 

Most of the customization features appear, not surprisingly, in the "Customize" dialog, available via the Tools menu.  You can re-configure all aspects of interaction with the program, such as the toolbars, menus (the main menu and popup or context menus), keyboard shortcuts, mouse double-clicks etc.  You can also set up HexEdit to easily run other software and change other user interface settings.

 

There are also some other useful ways to configure HexEdit in the Options dialog, which will also be covered in steps 5 and 6 of this tutorial.

 

 

Who Should Use this Tutorial?

 

It is for anyone who wants to configure the program to work the way they want it to, or for someone who uses HexEdit a lot and wants to save time for tasks they perform often.

 

It should take about 50 minutes to complete all steps of the tutorial.  However, the steps are independent so you can just do the ones you need or in the order you prefer.

 

 

What You will Learn

 

In this tutorial you will learn how to configure HexEdit in many ways including:

  • Create your own toolbar and add tools to it.

  • Configure an external tool to be quickly run from HexEdit.

  • Create custom keystrokes to accomplish tasks quickly.

  • Customize context menus to get instant access to tools you use often.

  • Set up color schemes and info-tips for specific purposes.

  • Configure the "Files of Type" list you see in standard Windows open/save to allow you to easily see just the files you are interested in.

  • Other miscellaneous customization tips.
