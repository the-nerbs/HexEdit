Step 4. Setting up External Tools

 

         

 

You can easily set up HexEdit to fire up other programs from the Tools menu.  This is useful if you need to quickly invoke related tools while viewing or editing files.

 

To show this we will set up the registry editor (regedit.exe) as a tool.

 

1. Open the Customize dialog and select the Tools tab.

2. Click the New tool button near the top of the window.

3. Enter the text "Registry Editor" as the description.

 

The description is the text that appears in the Tools menu.

 

You can use an ampersand (&) to indicate a character as a shortcut key.

 

Be careful not to conflict with any other shortcut keys used in the Tools menu - for example "R" is used for Record Macro near the end of the menu.

 

4. Change the description to "Registry &Editor".

5. In the "Command" box enter "regedit.exe".

 

You can enter the full path to a program or use the "..." (browse) button to find it.  In this case we know that regedit.exe will be found in the Windows system directory so we do not need to enter the full path.

 

 

6. Leave the "Arguments" and "Initial directory" boxes empty and close the dialog.

 

The arguments can include any command line argument needed by the program.  They can also include $(FilePath) which is replaced by the full path name of the active file (or left blank if there are no files open).  This is useful if you often use another tool to process the file you are working on.

 

7. Test that the registry editor is invoked when you select it from the Tools menu (or press Alt+T, E).
