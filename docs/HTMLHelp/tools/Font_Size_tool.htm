Font Size tool                              

 

 

 

Use this tool to select the size of the current display font.  Type in a font size or select a standard font size from the drop-down list.

 

Note that for Truetype fonts you can use any fonts size but only a limited number of standard font sizes are displayed in the list.  For other fonts (such as Courier) only the available sizes are displayed in the list.
