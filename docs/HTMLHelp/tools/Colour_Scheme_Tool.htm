Color Scheme tool                              

 

 

 

Use this tool to select a different color scheme for the active file.

 

To change the colors of a color scheme or add and delete color schemes use the Color Options page of the Options dialog.

 

Note that if you use one of the four standard color schemes (ASCII default etc) then when you change between different character set displays the color scheme will automatically change.  For example, if the current character set is ASCII (see Char Set ASCII) and the current scheme is "ASCII default" then changing to the EBCDIC character set (see Char Set EBCDIC) automatically changes to the "EBCDIC default" color scheme.
