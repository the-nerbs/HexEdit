Character Set EBCDIC command (View menu)

 

This displays the EBCDIC character set in the character area. This command is not available if the character area is not displayed (see Show Char Area or Show Both Areas).  Invalid EBCDIC characters and EBCDIC control characters are shown as a dot.

 

The options to show control characters in different ways (see Control Chars None, Control Chars Alpha, and Control Chars C/C++) are not available in EBCDIC mode.

 

This command also affects how string searches are performed (see Find Tool) and how text is copied to the clipboard (see Copy command). HexEdit will also beep if you try to type an invalid EBCDIC character into the character area.
