More (Code Pages) command (View/Code Pages menu)

 

Allows selection of any code page as the character set for display in the character area. This command is only available if the character area is displayed (see Show Char Area or Show Both Areas).

 

I opens the Document Display Options page to allow selection of any code page that is provided with your version of Windows, using the Code Page drop-down list.

 

You can also quickly select to display common code pages using commands for common code pages on the View/Code Page sub-menu.

 

You can quickly select common code pages using other commands on the View/Code Page sub-menu.
