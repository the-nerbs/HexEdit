Character Set Common Code Pages (View menu)

 

These commands allow you to switch to code page display and choose one of several common code pages in one step. To see all available code pages use the More ... Command to choose the code page from Document Display Options page.

 

The commands available are for these code pages:

  • 1252 Latin (ANSI) - this is the same as using Char Set ANSI

  • 65001 UTF-8 - this character set is derived from Unicode and is often used in web pages

  • 437 OEM - this should be the same as Char Set IBM/OEM

  • 500 EBCDIC International - this is similar to Char Set EBCDIC

  • 10000 Macintosh - as used on Macintosh computers

  • 1250 Latin - Central European

  • etc
