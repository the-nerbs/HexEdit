Auto Fit command (View menu)

 

This command toggles Autofit mode on and off. When autofit mode is on the columns for the window are adjusted when the window is resized so that the number of columns just fits across the window.

 

This can be used as a simple way of changing the number of display columns.  You can also set the number of columns explicitly as well as the column grouping and offset in the Window page of the Options Dialog.
