Toolbars: Custom Toolbar (View menu)

 

There are menu items corresponding to any custom toolbars that have been created (if any).  Use these commands (menu items) to toggle on or off the display of a custom toolbar. A check mark appears next to the menu item when the Toolbar is visible.

 

The Toolbars page of the customize dialog allows you to create and configure custom toolbars.  The Customize Commands page of the customize dialog allows you to change the buttons on a toolbar.
