Character Set ASCII+ANSI command (View menu)

 

This displays the ANSI character set which is a superset of the ASCII character set. This command is not available if the character area is not displayed (see Show Char Area or Show Both Areas).  Unlike for ASCII where bytes greater than 127 are shown as a dot, these show the ANSI characters which mainly consist of Western European language characters.

 

ASCII control character (characters less than 32) can be shown in various ways (see Control Chars None, Control Chars Alpha, and Control Chars C/C++).

 

This command also affects how string searches are performed (see Find Tool) and how text is copied to the clipboard (see Copy command).
