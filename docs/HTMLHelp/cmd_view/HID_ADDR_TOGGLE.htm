Decimal Addresses command (View menu)

 

Toggles between displaying decimal addresses and hex addresses for the active window. To allow you to more easily notice the difference, hex addresses are displayed with leading zeroes but decimal addresses are displayed with leading spaces.

 

If the “Digit Grouping in Addresses” global option is on (see Display Options page) then hex addresses are displayed in groups of 4 digits and decimal addresses are displayed with “thousands separators” (normally a comma separating groups of 3 digits in English speaking countries).
