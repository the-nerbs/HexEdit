Show Both Areas command (View menu)

 

This displays both the hex area and the character area to the right.

 

The character set displayed in the character area can be set with one of the character set commands (see Char Set ASCII, Char Set ANSI, Char Set IBM/OEM, and Char Set EBCDIC).  When the ASCII or ANSI character sets are displayed you can also change how control characters are displayed (see Control Chars None, Control Chars Alpha, and Control Chars C/C++).
