Toolbars: Standard Toolbar command (View menu)

 

Use this command to toggle on or off the display of the “Standard Toolbar”. The Standard Toolbar and the Edit Bar (see View Edit Bar command) are default toolbars which have buttons for many common commands in HexEdit. A check mark appears next to the menu item when the Standard Toolbar is displayed.

 

See Standard Toolbar for help on using the toolbar.  The Customize Commands page allows you to completely reconfigure all toolbars and even create new ones.
