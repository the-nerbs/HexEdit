Status Bar command (View menu)

 

Use this command to toggle the display of the Status Bar. The Status Bar contains several indicators which show the status of the display and certain keyboard states (CAPS and NUM lock). The left end of the status bar is used to display status messages. In particular, it describes the action to be executed by the selected menu item or depressed toolbar button.

 

A check mark appears next to the menu item when the Status Bar is displayed.

 

See Status Bar for help on using the status bar.
