Ruler command (View menu)

 

 

Use this command to toggle the display of the ruler in all hex views.  A check mark appears next to the menu item when the ruler option is on.

 

Which ruler is shown (hex or decimal) depends on whether addresses are shown in hex or decimal in the particular window - both rulers are shown if both types of addresses are shown.  How the ruler looks (placement of numbers, large ticks etc) can be configured in the Workspace Display Options.

 

The current cursor and mouse positions are also shown in the ruler (and address area), but this can be turned off in the Workspace Display Options.

 

The ruler also includes "markers" that show and allow you to quickly change display settings that are normally set in the Document Display Options page.  These are

  • Columns - the number of columns to display (this will be red if AutoFit mode is on)  

  • Grouping - how many byte are grouped together in columns

  • Offset - The column in which the first byte of the file is shown

To see the current numeric value of these settings hover the mouse over the "marker".  To change click and drag the marker.
