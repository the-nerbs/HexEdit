Help Topics command (Help menu)

 

Use this command to invoke the main Help window. On the left of the help window are four panes with tabs labelled Contents, Index, Search and Favorites.  Initially the table of contents is displayed.

 

The Contents pane displays headings and subheadings, like chapters in a book, that allow you to get an overview and quickly locate information if you know where it should be. You can use this to jump to major help topics.  Reference information on the HexEdit commands is not directly available through the Contents table but through links in various help pages such as each of the menu pages in the Reference section.

 

The Index pane allows you to look up subjects based on keywords stored in the help topics. Use this if you know what you’re looking for but don’t know where in the Contents hierarchy it is located.

 

The Find pane lets you search the entire help file for any word. This is a poor man’s index. Only use this if you find that the keywords in the Index pane are inadequate.

 

The Favorites pane allows you to save "bookmarks" for information that you think you may need to use again.
