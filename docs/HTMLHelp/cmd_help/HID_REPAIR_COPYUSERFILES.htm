Repair Copy User Files command (Help menu)

 

Use this command to copy HexEdit user files to a user's local data file folder (eg: C:\Users\<user>\AppData\Roaming\ECSoftware\HexEdit).

 

When you install HexEdit certain files are copied to your personal data folder (default templates, default toolbar images file, etc). If these files are not present HexEdit may not run properly - for example you may not be able to create a template or use/create custom toolbar images.

 

This command will fix this problem in case you have logged in as another user (not the one who was logged in when HexEdit was installed) or the files have been lost or corrupted somehow.
