Keyboard Map command (Help menu)

 

This displays a list or map of all the commands available in HexEdit and their shortcut keys.  The map shows the current keys if you have remapped your keyboard using the Customize Keyboard page of the Customize dialog.

 

The commands are categorised according to the HexEdit main menus.  You can view the commands in different categories by selecting from the Category drop-down list in the Key Map dialog.  There are some commands that do not appear in any menu -- these are available under the "All Commands" category.

You can print a list of all commands by selecting them and copying them to the clipboard, using the Copy button , then paste the text into Notepad or another editor or word-processor and print it.
