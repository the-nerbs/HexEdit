Tip of the Day command (Help menu)

 

Use this command to display the tip of the day, and to toggle whether tips are shown when the program starts up. This shows the same dialog that you see when you start HexEdit (if “Show Tips on Startup” is on).

 

By default “Show Tips on Startup” is on. This means that every time you run HexEdit a new tip is shown. These tips are taken sequentially from the (ASCII text) tip file HexEdit.tip. When all the tips have been shown the tips start from the beginning of the file again. (If you like, you can add your own tips to this file – one per line.)
