Repair Dialog Layout command (Help menu)

 

The modeless dialogs (Calculator, Bookmarks, Find etc) are very convenient as they are always available for quickly performing an operation.  However, they can get in the way of the display which is why HexEdit provides several mechanisms to avoid them obscuring the file you are working on.

  • You can hide the dialog by closing it, using the x button in the top right corner.

  • You can roll up the dialog, so only the title bar is visible, using the arrow button in the top right corner.

  • You can dock the dialog on the side of the window.  (See Toggle Dialog Dockability Command.)

  • You can make the dialog undocked (or floating) and move it to a different part of the screen or even a different monitor.

  • Lastly, HexEdit will move the dialogs out of the ways if they obscure the cursor position in the active file.

However, sometimes one of these mechanisms may make it hard to locate a dialog.  For example, the dialog may be rolled up or have been moved outside the visible area of the screen.  Also, sometimes registry settings become messed up and the dialog may become incorrectly docked and impossible to undock.

 

If you cannot find one or more modeless dialogs you can use this command to undock, unroll and move them all to the top-left of the HexEdit main window.
