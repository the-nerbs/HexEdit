Repair All command (Help menu)

 

This command removes all of files and registry settings that were created during and since installation.  It performs the equivalent of the Repair Settings command, and also deletes other configuration files used by HexEdit (bookmarks, recent file list etc - see below).  All registration information will also be removed so you will have to re-enter your activation code.

 

HexEdit use several disk files to store configuration information.  (This information is not stored in the registry as it could potentially be very large.)  These files are stored separately for each user in their local settings area (eg C:\Documents and Settings\andrew\Application Data\ECSoftware\HexEdit).  Currently there are three files:

  • Bookmarks - contains all bookmarks (file name, address, time created, time last accessed etc)

  • RecentFiles - contains info on each file opened (name, cursor/scroll/mark position, display options such as columns, highlights, associated template file, comments etc etc)

  • ToolbarImages.bmp - contains images for any custom toolbar icons you have created
