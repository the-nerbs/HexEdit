Template Global Options Dialog

 

 

This dialog allows you to set options that apply to the current template file.  This dialog is invokes by selecting the "Global Options" item from the context menu that appears when you right click the top item in the template tree view (see Using Templates).

 

Read-only

Makes the data fields of the file read-only by default, so that the user cannot modify the contents.  This setting may be overridden in specific DATA elements using the DATA dialog.

 

Big-endian

This sets the default byte order of certain types data fields in the template.  This setting can be overridden for specific elements using the DATA dialog..  It only applies to fields where byte order is relevant: numeric (integer and floating point), most date fields and Unicode strings.

 

Default Character Set

This is the default character set used to display strings.  It can be overridden in specific DATA elements.

 

Address of web site

This may contain a web address where the user can find out more information about the specific file format that the template is for.  Typically, you would enter the address where you found information you used to create the template, or the best such source available, on the Internet.

 

If this field is not empty then the "Template Web Site" button is active (found by default on the Navigation toolbar), and the user can click it to go to the web site.

 

Check

If you are connected to the Internet use this button to check the validity of the above web address.  It opens the default browser at that address.
