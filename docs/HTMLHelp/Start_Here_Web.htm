Overview

 

What is HexEdit?

HexEdit is a binary file editor.  It is used to view and modify binary files.

Who uses HexEdit?

During testing and debugging programmers find HexEdit invaluable for viewing and fixing binary files written or read by their programs.  Power users can use HexEdit to patch files or fix corrupt files such as documents and databases.

Benefits

HexEdit saves time by allowing you to track down and fix problems.  The intuitive user interface means you can work quickly to accomplish tasks that are difficult in other hex editors.

Powerful features such as keystroke macros mean you can easily do things that might otherwise take hours or days to write a program to do the same thing. Templates, configurable color schemes, property pages and search/compare tools make it easy to find what you are looking for.  Numerous user-interface refinements make HexEdit a joy to use.
