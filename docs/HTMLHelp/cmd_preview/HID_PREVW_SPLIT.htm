Split Window (Preview menu)

 

Displays the preview pane in a split window.

 

Currently the preview windows is only used to preview graphics (bitmap) files, such as .BMP, .JPG, .PNG, etc.

 

Once the preview is opened you many options for changing the view. For example, you can scroll using the mouse wheel (or click and drag the image) and zoom in/out by holding down the Ctrl key with the mouse wheel.
